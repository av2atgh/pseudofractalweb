# The pseudofractal web wants to be broken

Manuscript, figures, and reproduction scripts for an exactly solvable
calculation of the Ramsey community number of the Dorogovtsev–Goltsev–Mendes
pseudofractal scale-free web, under a plain Bernoulli stochastic block model
and a degree-corrected (configuration-null) model.

## Contents

Paper
- `manuscript.tex`   — Physical Review E manuscript (REVTeX 4.2)
- `refs.bib`         — bibliography
- `manuscript.bbl`   — processed bibliography (lets you skip the bibtex pass)
- `manuscript.pdf`   — compiled paper (6 pages)

Figures (PDF used by the manuscript; PNG for preview)
- `pseudofractal_figure.{pdf,png}` — Fig. 1: P(split) and |log R| vs N,
  plain vs degree-corrected
- `partition_branch.{pdf,png}`     — Fig. 2: the recursive branch cut
- `partition_hubleaf.{pdf,png}`    — Fig. 3: the hub–leaf (core–periphery) cut
- `partition_3way.{pdf,png}`       — Fig. 4: the symmetric three-community partition

Scripts
- `pseudofractal_verification.py` — reproduces every table and slope from
  scratch; checks the closed-form counts against direct construction of the web
- `main_figure.py`                — regenerates `pseudofractal_figure.pdf`
- `partition_figures.py`          — regenerates the two partition figures
  (optional generation argument, e.g. `python3 partition_figures.py 4`)

## Reproduce

Verify the numbers:
```
pip install networkx mpmath numpy matplotlib scipy
python3 pseudofractal_verification.py
```

Regenerate the figures:
```
python3 main_figure.py
python3 partition_figures.py 4
```

Compile the paper (requires a TeX distribution with REVTeX 4.2):
```
pdflatex manuscript ; bibtex manuscript ; pdflatex manuscript ; pdflatex manuscript
```

## Key results

- Plain Bernoulli SBM: the branch cut is unpreferred while small and breaks at
  r_kappa = 1095 nodes, with log R ~ (ln3 − (2/3)ln2) N.
- Degree-corrected (configuration null): the same cut survives, breaking at
  r_kappa = 42, with log R ~ (2 ln3 − (4/3) ln2) N — exactly twice the plain
  slope, prior-independent.
- Degree correction reverses which cut is preferred: it demotes the hub–leaf
  split (a degree artefact) and elevates the recursive branch cut (genuine
  hierarchical community structure).
- The web is self-similar, so the best description is a nested hierarchy: the
  degree-corrected evidence is maximised at K_opt = 3^floor(T/2) ~ sqrt(N)
  communities of ~sqrt(N) nodes each (each level adds ~2 ln3 * N).
